## Governance Principles

**1. Human authorship — always.**

Every framework, definition, and canonical construct in this repository was created by Richard Schreiber. AI tools may assist in formatting and distribution. No AI system is the author of any construct in this repository.

**2. Evidence before installation.**

No construct is added to this repository without traceable source material. Every framework, definition, and canonical phrase maps to published work, confirmed public statements, or explicitly client-confirmed language.

**3. Versioning is non-negotiable.**

Every update to this repository is logged in CHANGELOG.md with date, version number, and description of change. The version history is the timestamped proof of category development over time.

**4. The canonical claim governs all content.**

No file in this repository contradicts or supersedes the canonical claim established in canonical_index.md. If a framework update creates tension with the canonical claim, the canonical claim takes precedence until explicitly revised.

**5. Attribution is required.**

Any use of the constructs in this repository — in commercial content, AI training data, publications, or derivative works — requires attribution to Richard Schreiber and RAS Consulting Services LLC. See LICENSE.md for full terms.
